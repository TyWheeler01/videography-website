# Wedding Videography Portfolio Site

This is the official website for my Wedding Videography, built as a modern, high-performance static site and deployed on Cloudflare Pages.

## Features

* **Responsive Design:** Fully optimized for desktop and mobile devices.
* **High-Performance Video:** Utilizes Cloudflare Stream for adaptive bitrate (HLS) playback, to ensure fast, smooth and cross-browser compatible video display.
* **Static Deployment:** Deployed using Cloudflare Pages for speed, security and global CDN coverage.
* **Professional Workflow:** Managed with Git and GitHub to ensure a reliable CI/CD pipeline.

## Technology Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Video Hosting:** Cloudflare Stream (HLS)
* **Deployment:** Cloudflare Pages (CI/CD from GitHub)
